# Admin Panel

## [Admin Panel link](https://food-app-admin-mu.vercel.app/)

<img width="1266" alt="Admin Panel" src="https://github.com/Jiayulllll/food-app/blob/master/admin/public/admin.png">

### Click the Add items

you can update menu by adding food image, name, description,category and price.

**!!! Notice !!!:** due to the server limit, it will take a little bit long time to upload image to the database.

### Click the List items

you can see all the items in the menu, and you can delete food from list.

### Click the Orders

you can see all the orders,you can change the order status

### Click the Add Delivery Postcode

you can delete and add the postcode that are available deliver

## For the localhost user

In the /food-backend, you can run:

### `npm run server`

In the /food-front, you can run:

### `npm run dev`

Runs the app in the development mode.\
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

In the /admin, you can run:

### `npm run dev`

Open [http://localhost:5174](http://localhost:5174) to view it in your browser.

to update the menu,
to change order status,
to add/delete the delivery postcode.
